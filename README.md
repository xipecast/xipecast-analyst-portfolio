## Key Findings

### Titanic
- Women survived at **74.2%** vs men at **18.9%**
- Poor males in 3rd class had only **13.5%** survival rate
- Gender was stronger predictor than class or age

### House Prices
- Location gap = **\$191,437** between best/worst neighborhoods
- Quality rating **7 = luxury threshold**—prices jump dramatically
- Post-**1980 construction** = quality inflection point

### House Prices ML
- Achieved **90.9% R²** accuracy predicting house sale prices
- **OverallQual** = strongest predictor, more important than ALL other features combined
- Improved baseline model by **6.5%** through hyperparameter tuning
- Compared Linear Regression, Random Forest, and XGBoost

### Chinook Music Store
- **USA = #1 revenue** country (\$523 = 2x Canada)
- **Rock = dominant genre** (\$826 revenue)
- **Iron Maiden = top artist** (\$138 revenue)
- All 3 sales agents perform similarly per transaction
